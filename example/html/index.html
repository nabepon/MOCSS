MOCSS サンプル

概要

このページはCSS設計【MOCSS】のごく小さなサンプルページです。
MOCSSについての概要は こちらのページ を参照してください。

ディレクトリ構成

src/style/common
共通スタイル用ディレクトリ。_index.scssが他ディレクトリからのインターフェースになっています。
src/style/models
Model用ディレクトリ。mixin以外は禁止のディレクトリです。
src/style/packages
Package, Component, Pageのディレクトリ。規模が大きい場合は、Packageをさらにディレクトリにして分割します。今回は小さいので1Package 1ファイルです。
src/template
ejsのテンプレートが入っています。

環境

このサンプルページでは以下のツールを使用しています。

  • gulp
  • sass
  • postcss
  • ejs
  • Browsersync

実行方法

GitHubからこのプロジェクトをcloneしてきます。
npm install を実行します（最初の1回のみ）。
npm run dev を実行します。
srcがビルドされ、exampleに吐き出されます。
example/html/index.html を開くと確認できます。
